## MODELO GENÉRICO DE SISTEMA

ENTRADA ---- SISTEMA ---- SAÍDA

## Definição genérica dos seis componetes:

Objetivo: Diz respeito a finalidade a qual o sistema foi criado.

Entradas: A matéria-prima que inicia o processo de transformação, ou seja, o material, a energia ou os dados que iniciam o processo. 

Processamento: Transformação da matéria-prima(entrada) e um proiduto, serviço ou resultado (saída).

Saídas: Resultados do componente processamento, pode ser produto, serviço ou informação, deve ser coerente com o objetivo do sistema.

Controles e avaliações: Verificam se todos os componetes estão coerentes com os objetivos estabelecidos.

Retroalimentação/feedback: É um instrumento de controle que visa garantir que a finalidade do sistema esteja sendo atingida com sucesso, pode ser considerada uma nova entrada para o sistema.

## SISTEMA DE INFORMAÇÃO (SI)

O sistema da informação é um conjunto componentes relacionados entre si que coletam, recuperam, armazenam, processam ou distribuem informaçõess destinadas ao apoio de decisões em uma organização. Além disso o SI pode auxiliar gerentes e colaboradores a identificar problemas, visualizar assuntois complexos e criar novos produtos e soluções.
Os sistemas de informação necessitam de uma infraestrutura para o seu funcionamento, hardware, software, banco de dados e rede de computadores.

INFORMAÇÕES DE ENTRADA => SOFTWARE - HARDWARE - REDE DE DADOS => INFORMAÇÕES DE SAÍDA

#### O Software funciona como o cérebro do computador, porque é ele quem dá as intruções e organiza como o corpo (Hardware) deve agir. Porém diferente do cérebro humano qur também armazena memórias o software, não guarda os dados sozinhos - ele usa o "corpo" (HD,SSD, memória RAM) para isso. A rede de computadores funciona como o sistema nervoso do corpo. Ela conecta diferentes partes (computadores, servidores, dispositivos) e permite que as informações circulem de um ponto a outro. Sem a rede, cada computador ficaria isolado, como um órgão sem nervos para se comunicar com o resto do corpo.

* Hardware é o corpo ( os ossos, os musculos, e os orgãos).
* Software é o cérebro que envia os comandos e organiza as aações.
* Banco de dados = memória de longo prazo (guarda informações).
* Rede de computadores = sistema nervoso (faz a comunicação entre diferentes partes).
